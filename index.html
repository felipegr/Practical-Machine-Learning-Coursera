Next step was to set a seed, separate my data into a training set and a test set and build the model. I opted for the gbm method, because I thought it would be useful in this classification problem. I chose to use the “train” function options to preprocess the data using the PCA method and to make a 10-fold cross validation.

set.seed(1235)

inTraining <- createDataPartition(modified_training$classe, p = 0.75, list = FALSE)
training <- modified_training[inTraining, ]
testing <- modified_training[-inTraining, ]

modelFit <- train(training$classe ~ ., method="gbm", data=training,
                  preProcess="pca", trControl = trainControl(method = "cv",
                                                             number = 10,
                                                             repeats = 10))
